HTTP POST

POSTs a JSON array of readings to any HTTP endpoint. Compatible with InfluxDB v2 (/api/v2/write), Grafana, custom servers.

Sensor.Community

Sends PM2.5/PM10 and environmental data to sensor.community. Register your device at the site first to get a chip ID. The device ID is your ESP32 MAC-derived ID (shown in Device settings). Send interval must be ≥ 145 seconds (API requirement).

🗺️ openSenseMap

Sends readings to opensensemap.org. Create a senseBox at the site, then copy the Box ID, access token and sensor IDs below.
