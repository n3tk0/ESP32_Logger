💧 Flow Meter

Primary pulse-counter flow sensor (hardware pin fixed in Hardware). Treated as a dedicated sensor because its pulses drive the state machine.

📏 Calibration

Number of flow sensor pulses per liter. Check your sensor datasheet (e.g., YF-S201 = 450).

Fine-tune measured volume. 1.0 = no adjustment. Adjust up/down based on comparison with a known volume.

⏱️ Timing

Idle time before logging.

Delay on first loop (for delayed fill).

🧪 Test Mode

WiFi Pin as LED output. Solid = IDLE / Blink = Water running.
